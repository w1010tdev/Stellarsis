# Markdown 与 LaTeX 快速入门（简明）

以下内容用中文说明如何在 Stellarsis 的帖子/聊天中写 Markdown 与 LaTeX（适合多项式与微积分类数学表达）。

1) 基础 Markdown

- 段落：直接写文本并空行分段。
- 换行（强制换行）：在行尾添加两个回车。
- 标题：使用 #，例如 `# 一级标题`、`## 二级标题`。
- 加粗：`**加粗文本**`。
- 斜体：`*斜体文本*` 或 `_斜体_`。
- 列表：
  - 无序列表：以 `- ` 或 `* ` 开头。
  - 有序列表：`1. ` `2. ` 等。
- 代码：行内代码使用反引号，例如 `` `x = 2` ``；代码块使用三个反引号：

````
```
def f(x):
    return x**2
```
````

- 插入图片（Markdown格式）：
  - 语法： `![替代文字](/static/uploads/USER_ID/filename.png)`
  - 系统上传后会返回一段 `![alt](url)` 的 Markdown，点击复制即可粘贴进编辑器。

2) LaTeX（数学公式）基础

Stellarsis 支持内联与块级 LaTeX 公式（基于常见的渲染器）：

- 行内公式：用 `$ ... $` 包裹，例如 `$x^2 + y^2 = z^2$`。
- 块级公式：用 `$$ ... $$` 包裹，写在单独一行，用于较大或独立的公式：

  ```
  $$
  \int_0^1 x^2 \,dx = \frac{1}{3}
  $$
  ```

注意：在聊天短消息中尽量使用行内公式避免换行过多。

3) 常见多项式示例（Markdown + LaTeX）

- 多项式定义（行内）：
  - 文本：`多项式 $p(x)=3x^3-2x+1$ 在区间 [0,1] 上的值为...`。
- 展示多项式（块级）：
  ```
  $$
  p(x) = 3x^3 - 2x + 1
  $$
  ```
- 因式分解（块级）：
  ```
  $$
  p(x) = (x-1)(3x^2+3x-1)
  $$
  ```

4) 微积分常用输入示例

- 导数（行内）：`$f'(x) = 3x^2 - 2$` 或 `$\frac{d}{dx} f(x) = 3x^2 - 2$`。
- 不定积分（块级）：
  ```
  $$
  \int 3x^2 - 2 \,dx = x^3 - 2x + C
  $$
  ```
- 定积分：
  ```
  $$
  \int_a^b x^2 \,dx = \left[\frac{x^3}{3}\right]_a^b = \frac{b^3-a^3}{3}
  $$
  ```

5) 小技巧与常见错误

- 如果公式中使用反斜杠（例如 `\frac`、`\int`），请在编辑器中像上面示例一样直接写 `\`，编辑器会正确渲染为 `\`（在原始 Markdown 文本中写 `\` 表示 LaTeX 的 `\`）。
- 避免在公式内直接插入未经转义的 Markdown 元字符（例如 `*`、`_`），如需使用，请用反斜杠转义或用 LaTeX 命令完成样式。
- 长公式建议使用块级公式 `$$...$$`，可提高可读性。

6) 快速示例（整段可复制）

````markdown
这是一个多项式示例：

$$
p(x)=3x^3-2x+1
$$

导数为 $p'(x)=9x^2-2$，在 $[0,1]$ 区间上的定积分为：

$$
\int_0^1 p(x)\,dx = \left[\frac{3}{4}x^4 - x^2 + x\right]_0^1 = \frac{3}{4} - 1 + 1 = \frac{3}{4}
$$
````

把上面内容粘贴到发帖编辑器即可看到 Markdown + LaTeX 渲染效果。
